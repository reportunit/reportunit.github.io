ReportUnit

    Features

    It uses stock XML reports from NUnit, MSTest and Gallio and converts them into beautiful HTML reports for folder and file levels.

    1 API for 3 Test Runners

    Single tool to convert XML files from a variety of test-runners with simple console commands.

    Easy to Use

    Extremely easy to use. Simply execute the command in console to generate an HTML output.

    Beautiful Reports with Dashboard

    Creates beautiful folder and file reports with dashboards.

    Free for Commercial Use

    ReportUnit is completely free for both personal and commercial use.
