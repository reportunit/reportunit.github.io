What It Is

ReportUnit uses stock XML reports from NUnit, JUnit, TestNG and Gallio and converts them into beautiful HTML reports for folder and file levels.

1 API for 4 Test Runners

Single tool to convert XML files from a variety of test-runners with simple console commands.

Easy to Use

Extremely easy to use through a well-defined console command structure.
